# Logo Placeholder

To add your Shiba Solana logo:

1. Create or design your logo (recommended size: 512x512px or larger)
2. Save it as PNG format with transparent background
3. Name it `logo.png`
4. Place it in this `assets` folder

## Recommended Logo Specifications:

- **Format**: PNG with transparent background
- **Size**: 512x512px minimum (for high-resolution displays)
- **File name**: logo.png
- **Design tips**: 
  - Make it recognizable at small sizes
  - Use colors that match your brand (#ff6b00 is the primary color)
  - Consider adding a Shiba dog theme
  - Include Solana elements if desired

## Additional Images You Can Add:

- `hero-image.png` - Large image for hero section
- `about-image.png` - Visual for about section  
- `favicon.ico` - Browser tab icon (16x16, 32x32, 48x48)
- Social media graphics
- Team member photos
- Partnership logos

Once you add your logo, it will automatically appear in the navigation bar!
